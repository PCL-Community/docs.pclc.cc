PCL 目前支持正版登录、离线登录、统一通行证和 Authlib-Injector 登录，也可以单独为某一版本指定登录方式。本篇将对此提供简单的教程。

| 步骤                            | 图片                                                             |
|-------------------------------|----------------------------------------------------------------|
| 1. 点击启动页面下方的 "版本选择"。          | ![图片](/contents/23c5f07d81e3ca4a0f2208d45f4b894c565437509.png) |
| 2. 右键点击需要修改的游戏版本。             | ![图片](/contents/4f6dbbe781eecc06ba8151ef1acf0bcf565437509.png) |
| 3. 点击左侧 "设置" 按钮。              | ![图片](/contents/8a1e28c4e649287b701366e90dd6b8af565437509.png) |
| 4. 滑动至下方，在 "服务器" 中选择登录方式即可切换。 | ![图片](/contents/d14d60b1f634bf4bc56baa2be5dc7abc565437509.png) |
