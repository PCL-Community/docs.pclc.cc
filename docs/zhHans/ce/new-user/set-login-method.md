# 指定登录方式

::: danger 💀 过时文档
该页面内容已被标记为过时，并可能与当前实现不一致。
:::

> **作者**: XiaoFans、一闪

PCL 目前支持正版登录、离线登录、统一通行证和 Authlib-Injector 登录，也可以单独为某一版本指定登录方式。本篇将对此提供简单的教程。

1. 点击启动页面下方的 "版本选择"。

2. 右键点击需要修改的游戏版本。

3. 点击左侧 "设置" 按钮。

4. 滑动至下方，在 "服务器" 中选择登录方式即可切换。
