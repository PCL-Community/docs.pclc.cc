在部分设置、自定义主页和帮助中，可以使用类似 `{time}` 的替换标记，它可以动态替换设置和文件中的某些部分。

例如，在 设置 → 启动选项 → 游戏窗口标题 中输入 `现在是 {time}`，在启动游戏后就能在游戏窗口看到当前时间。

你也可以在 XAML 中直接使用替换标记，例如：

```xaml
<TextBlock Text="现在是 {time}" />
```

就会被自动替换为：

现在是 <CurrentTime />
