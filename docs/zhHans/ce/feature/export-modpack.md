# 导出整合包

PCL CE 内置了整合包导出功能，可以快速将当前游戏版本打包成整合包，并分发给朋友或玩家使用。

PCL CE 会自动检测当前版本中可以导出的内容，并根据已安装的模组显示相关选项，尽量减少手动整理文件的步骤。

整合包作者还可以通过配置文件精准控制导出内容，或重复使用之前保存的导出方案。相关内容见下文的“使用配置文件导出”。

## 一、基础导出方法

| 步骤                                      | 图片                                                             |
|-----------------------------------------|----------------------------------------------------------------|
| 1. 在 PCL 的“启动”页面中选择要导出的游戏版本，然后点击“版本设置”。 | ![图片](/contents/fa623fdb189cd5d32ab362d0d3a1cd58506713078.jpg) |
| 2. 点击“导出”，进入整合包导出页面。在这里可以自定义整合包名称和版本号。  | ![图片](/contents/73abe4dc45777672e846f8fbe796fbdf506713078.jpg) |
| 3. 选择需要导出的内容后，点击“开始导出”，即可完成导出。          | ![图片](/contents/f4fb19a6c5dfda5c44dc5ed4929bc2e1506713078.jpg) |

::: tip 提示
一般情况下，为了保证整合包可以正常运行，至少需要导出“模组”选项，以及其下方的“整合包重要数据”和“模组设置”。其他内容可以按需选择。

勾选“PCL CE 启动器程序”及其下方的“PCL CE 个性化内容”后，整合包内会包含 PCL CE 程序文件，以及你在“设置 → 个性化”中配置的相关内容。
:::

## 二、调整整合包导出形式

PCL CE 默认以 Modrinth 格式导出整合包。你可以通过以下两个选项调整导出方式。

### 1. 打包资源文件，以避免在导入时下载

启用后，模组、资源包、光影包等文件会被直接放入整合包中。这样在导入整合包时，无需再联网下载这些文件。

该选项会增大整合包体积，建议仅在无法稳定连接 CurseForge 或 Modrinth 时启用。

### 2. Modrinth 上传模式

启用后，PCL CE 会生成完全符合 Modrinth 要求的整合包文件，适用于需要上传至 Modrinth 的整合包。

## 三、使用配置文件导出

PCL CE 为整合包作者提供了配置文件功能，可以通过配置文件精准控制导出范围。

初次使用时，可以先在某个版本的导出页面中点击“保存配置”，导出一份配置文件。配置文件内包含详细注释，可根据注释修改导出内容。

修改完成后，再次导出整合包时，只需点击“读取配置”，选择对应的配置文件，即可将该配置快速应用到当前整合包导出中。

::: tip 提示
普通玩家通常只需阅读上方的基础导出方法即可。以下内容更适合整合包作者、服务器客户端制作者，或了解相关基础知识的玩家阅读。
:::

## 四、整合包分发与自动安装

你目前使用的是 PCL 社区版。你可以将其发给其他人，也可以用于制作整合包或服务器客户端，无需额外申请授权。

若整合包文件名为 `modpack.zip` 或 `modpack.mrpack`，并且与 PCL CE 的 exe 文件位于同一文件夹下，那么 PCL CE 会在打开时自动安装该整合包。

在导出时勾选“PCL CE 启动器程序”后，PCL CE 会自动按照上述格式进行打包。

无需担心正版账号等敏感信息，它们不会随着整合包一起发放。

## 五、整合包中的启动器设置

整合包作者可以根据需要调整启动器的个性化内容、更新策略和功能入口。

### 1. 自定义启动器主页

你可以在“设置 → 个性化 → 主页”中自定义启动器主页。

也可以将主页 XAML 文件上传到服务器，并通过“联网更新”选项实现主页动态更新。

::: warning 注意
PCL 社区版不支持主页联网更新功能。需要使用该功能时，请使用主线版本。
:::

### 2. 设置启动器更新方式

你可以在“设置 → 启动器 → 系统”中修改启动器的更新方式。

能够通过 QQ 群等方式通知到所有整合包玩家时，可以关闭启动器更新。

无法通知到所有玩家时，建议至少选择“仅在有重大漏洞更新时显示提示”，避免玩家因无法收到重大漏洞更新而影响正常游戏。

### 3. 设置公告推送方式

启动器公告包含紧急漏洞通知、PCL CE 活动通知、新功能投票等推送弹窗。

需要屏蔽 PCL CE 活动通知时，可以降低公告推送等级。没有特殊需求时，推荐设置为“显示所有公告”。

### 4. 隐藏不需要的功能入口

你可以在“设置 → 个性化 → 功能隐藏”中关闭不需要的功能入口，例如下载、个性化设置等页面。

## 六、为整合包附带 Java

需要让整合包自带某个版本的 Java 时，可以直接将 Java 放在版本文件夹中：

```text
versions/版本名
```

PCL CE 会自动搜索并优先使用放在版本文件夹内的 Java。
